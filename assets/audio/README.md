# Audio Assets

This directory should contain the audio files for the meditation app.

## Required Files

### Nature Sounds (assets/audio/nature/)

- `rain.mp3` - Rain sound loop
- `ocean.mp3` - Ocean waves sound loop
- `forest.mp3` - Forest ambiance sound loop

### Music (assets/audio/music/)

- `piano.mp3` - Calm piano music loop
- `ambient.mp3` - Ambient music loop

### Sound Effects

- `chime.mp3` - Completion chime sound

## Audio Specifications

- Format: MP3
- Bitrate: 128kbps
- Sample Rate: 44.1kHz
- Duration: 5+ minutes (for looping tracks)
- Volume: Normalized to -14 LUFS

## Sources

You can find royalty-free meditation audio from:

- Pixabay Audio Library
- Freesound.org
- YouTube Audio Library
- Incompetech.com

## Note

The app is currently configured to load these files. You need to add actual audio files with these exact names for the audio features to work.
